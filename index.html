公式=IF(A2=””,0,1)释义：

如果A2单元格是空白，条件成立，此时的结果将会是0（即：FALSE），对于数据有效性而言，条件将不成立，此单元格不能输入数值。

如果A2单元格不是空白，条件不成立，此时的结果将会是1（即：TRUE），对于数据有效性而言，条件将成立，此单元格可以输入数值。

日语笔记

发表于2021年3月21日由admin

見ていた

动词见る的连用形见＋ていた。ていた表示动词的过去进行时，是动作主体在过去某段时间内做了什么事。

残さず

“不留下”“不剩下”的意思，然后是它的连用型
残す➡︎残さない➡︎残さず
す变成否定是さない，然后变成连用型是さず。 加ず只是后面还要加某个用言，做得变形

続いてる

动词原形“続く”，这里是进行时态，表示“还在继续着”。

Python下载URL图片

发表于2021年3月21日由admin

使用第三方库requests：

# coding: utf8
import requests

def download_img(img_url, api_token):
    print (img_url)
    header = {"Authorization": "Bearer " + api_token} # 设置http header，视情况加需要的条目，这里的token是用来鉴权的一种方式
    r = requests.get(img_url, headers=header, stream=True)
    print(r.status_code) # 返回状态码
    if r.status_code == 200:
        open('C:\\Users\\cloudoxou\\Desktop\\img.png', 'wb').write(r.content) # 将内容写入图片
        print("done")
    del r

if __name__ == '__main__':
    # 下载要的图片
    img_url = "http://www.baidu.com/some_img_url"
    api_token = "fklasjfljasdlkfjlasjflasjfljhasdljflsdjflkjsadljfljsda"
    download_img(img_url, api_token)

 

这里是用GET方法来直接请求图片URL，URL随便编的，其中加了一个header，是为了演示需要鉴权的情况，有些网站是需要登录才能访问的，这里的”Authorization”: “Bearer “就是一种鉴权方式

绕过WAF

发表于2021年3月20日由admin

发paper这种事，我作为本科生也无法免俗，于是想着拿自己多年和WAF对抗的经验，看看能不能搞搞自动过WAF的技术。 闷头搞肯定是不行的，于是网上搜点相关paper读，同时也搜集乌云先知上的各种案例看看能不能找到点理论化的门道，然后就发现这些paper不太靠谱了..

乌云和先知上面在payload的层面绕过WAF的案例，大部分是用WAF不认识的冷门函数和冷门语法冷门运算符冷门空字符来实现，还有一部分靠着花式注释让WAF误把注释的当做没注释，没注释的当做注释扰乱分析，一旦发现了一个点，WAF就绕过去了

但是各种论文里面的方式是，内置了一堆变异策略和替换策略，用各种算法选取它们，然后叠加起来绕过WAF。 在论文中，变异和替换策略是一来就给出来的，相当理所当然，而文章的重点就会放到各种神奇的策略选取算法

但真正的重点，实际上是貌似理所当然的变异和替换的策略，合适的变异策略和替换策略就能轻松瓦解WAF的防护，而不合适的策略再怎么叠加都无济于事。 这些策略是经验丰富的黑客根据对句法的理解和对WAF构造的理解一点点摸索出来的，而我所找到的paper没有一篇在讨论如何找到策略

lnmp中imagick模块的安装

发表于2021年2月21日由admin

安装ImageMagick图像软件

cd ImageMagick
./configure
make && make install
ImageMagick下载地址在
安装imagick php扩展插件
    imagick插件工作需要ImageMagick软件的支持，所以，必须要先安装ImageMagick，否则会出现报错等一系列问题。
    imagick插件是一个可以供PHP调用ImageMagick功能的扩展模块。使用这个扩展可以使PHP具备和ImageMagick相同的功能。装了ImageMagick图像程序后，再装PHP的扩展imagick插件，才能使用ImageMagick提供的api来进行图片的创建与修改、压缩等操作。因为他们都集成在了imagick这个PHP扩展中。
 imagick-2.3.0.tgz版本太低了，不支持php5.5,
需要ImageMagick-6.7.9-9.tar.xz的支持，因此，注意先后顺序。

imagick扩展包地址：

https://pecl.php.net/package/imagick

下载imagick最新稳定版

wget https://pecl.php.net/get/imagick-3.4.4.tgz
tar xf imagick-3.4.4.tgz
cd imagick-3.4.4
/usr/local/php/bin/phpize

根据自己phpize的位置来运行即可（不知道phpize在哪的直接find / -name phpize查找），运行完提示下面信息，然后编译

Configuring for:
PHP Api Version: 20160303
Zend Module Api No: 20160303
Zend Extension Api No: 320160303

编译扩展模块

./configure --with-php-config=/usr/local/php/bin/php-config
make
make install

执行完成后会显示扩展模块编译到的目录/usr/local/php/lib/php/extensions/no-debug-non-zts-20160303/

修改php.ini添加扩展

vi /usr/local/php/etc/php.ini

extension=/usr/local/php/lib/php/extensions/no-debug-non-zts-20160303/imagick.so #增加项

重启网站

使用Nginx屏蔽/禁止IP访问网站

发表于2021年2月20日由admin

在/usr/local/nginx/conf/nginx.conf配置文件中，找到server然后在server”{}”，在这个大括号内加入

deny IP地址是限制某IP地址访问；

allow IP地址是只允许某IP地址访问；

    server {
        listen       8080;
        #server_name  localhost;
        server_name  182.243.41.257;
        root   "D:/qiPaiBackend/backend/web";
        error_page   500 502 503 504  /50x.html;
        # 配置ip限制策略
		# 允许部分ip访问
		allow 182.243.41.257;
		allow 127.0.0.1;
		 # 禁止其余ip访问
		deny all; 
        location = /50x.html {
            root   html;
        }
        location / {
            index index.php index.html error/index.html;
            try_files $uri $uri/ /index.php$is_args$args;
        }
        
        location ~ \.php(.*)$ {
            fastcgi_pass   127.0.0.1:9000;
            fastcgi_index  index.php;
            fastcgi_param  SCRIPT_FILENAME  $document_root$fastcgi_script_name;
            fastcgi_connect_timeout 600;
            fastcgi_read_timeout 600;
            fastcgi_send_timeout 600;
            include        fastcgi_params;
        }
    }

adb命令

发表于2021年2月16日由admin

重启到Recovery界面

adb reboot recovery

重启到bootloader界面

adb reboot bootloader

ADB在计算机和设备之间上传/下载文件，将电脑中的apk软件安装至android设备速度非常快：
1、下载文件
语法：adb pull [文件名] /源路径/
例如：
adb pull blade_keypad.kl /system/usr/keylayout/
adb pull /sdcard/mp3/1.mp3

2、上传文件
语法：adb push [文件名] /目标路径/
例如：
adb push qwerty.kl /system/usr/keylayout/
adb push 1.mp3 /sdcard/mp3/

3、安装程序
语法：adb install -r [/路径/软件名.apk]
例如：adb install -r com.android.vending.apk
adb install -r com.android.vending.apk

安装：adb install -r com.android.vending.apk
